How I Built My First Portfolio Website

By Avinash | Published on August 2, 2025

Building my first portfolio was a major step in my web development journey. It helped me understand the core principles of HTML, CSS, and the importance of design.

What I Learned

  • How to structure content using semantic HTML
  • Basic styling with the CSS Box Model
  • Organizing files and folders in a project

I began with planning the sections: About Me, Projects, and Contact. Then I used HTML5 tags like <header>, <nav>, and <section> to structure the content properly.

What’s Next?

I plan to upgrade my site using React, improve responsiveness, and eventually deploy it online with a custom domain.
